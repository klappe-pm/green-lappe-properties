## Reading Order

1. Read the 2-pager.
2. Review the deck.
3. Use the market deep dive as the support memo.
4. Use the source and corpus indexes to trace claims back to project evidence.

## Working Premise

Megan and Kevin are already aligned on the pain points and the broad opportunity. These materials therefore do not restart the research. They convert the corpus into a decision package: what is true enough to act on, what is still assumption-bound, what the first 30/60/90 days should prove, and what a partner conversation needs to decide.
