# Key decisions

- Did not buy domains, configure DNS, or publish website content.
- Treated evidence storage as the next unblocked prerequisite because checkout evidence will contain sensitive account, billing, or personal information.

- Added pointer formatting so repo docs can reference evidence without share URLs, credentials, personal contact details, or sensitive screenshots.

# Risks and open questions

- Partners still need to choose the actual controlled evidence container before checkout.
- Sensitive evidence storage may need counsel, accounting, insurance, or privacy input depending on the record type.
- Domain availability can change before checkout.
- The actual registrar cart price, renewal price, taxes, and privacy settings still need to be captured by the account owner immediately before purchase.
- The registrar/DNS pattern remains unresolved: Cloudflare-only for simplicity, or independent registrar plus Cloudflare DNS for separation.

- Do not buy domains without explicit partner approval and account-owner action.

- Do not buy speculative backup domains without explicit approval.
