# Summary

Created a local-only Green Lappe market opportunity analysis memo and a supporting Excel financial model for property-management market entry in King and Snohomish counties. The memo frames the opportunity from problems and unmet needs through personas, pain scope, county-level financial sizing, TAM/SAM/SOM, moat, and financial implications for slide-deck input. The workbook models a conservative 5-year base case using the user's Year 1 assumption of 5 properties at 75% of a full operating year, plus TAM/SAM/SOM and scenario comparison sheets. No web search was used; inputs came from repository files and explicitly labeled analyst assumptions.

# Key decisions

- Used the Effort 2 CSV-derived small-owner rollup for the workbook TAM math because it ties directly to the ZIP-level project dataset.
- Treated Snohomish County as the launch/proof beachhead and King County as the scale/moat market, matching the launch memo even though King ZIPs dominate the PM sequencing score.
- Modeled no maintenance-markup revenue to preserve the transparent-fee positioning in the brand and persona research.
- Kept the base case below the repo's bootstrapped SOM scenario so the first model is a quality-first entry case rather than a funded-growth case.

# Risks and open questions

- The model excludes founder opportunity cost and acquisition capital, so it should not be read as a full company P&L.
- Fixed operating costs and direct ops costs are analyst placeholders; replace them after the first operating quarter with actual time and vendor data.
- The workbook uses local repository values only; source documents contain some differing rent-roll estimates, and the memo explicitly flags the conservative TAM memo versus Effort 2 CSV difference.
- The working tree had many pre-existing unrelated modifications, deletions, and untracked files before this session; they were not staged intentionally.
