# Key decisions

- Framed the partner conversation as a 90-day proof sprint decision rather than a full launch decision.
- Used the local corpus only and kept the deck/memo narrative focused on what Kevin and Megan already broadly agree on: owner pain, tenant pain, operator fragmentation, and a staged Snohomish-first opportunity.
- Preserved the difference between broad TAM, practical property-management SAM, and proof-sprint SOM so market size does not imply near-term capacity.
- Treated FCC as a later trust/moat wedge, not the day-one operating product, matching the prior passoff constraint.
- Built the style guide around restrained Pacific Northwest greens/blues with chart-first slide rules and source-footnote discipline.
- Built the corpus index as a pointer system, not a replacement for reading source docs; it records paths, frontmatter, headings, hashes, classification, and query guidance for the next Codex session.

# Risks and open questions
